Chapter 14 – IT Essentials

01. Which type of printing option should be implemented to allow a user to change settings for each document sent to the printer?

  • default ;
  • general ;
  • global ;
  • per-document ;
  • pre-script.

02. A technician is installing a new inkjet printer. The printer has successfully printed test pages. Which task also verifies the functionality of an inkjet printer after installing the printer driver?

  • sending print jobs for at least two minutes to verify the printer speed;
  • configuring the highest resolution before printing to check the printer memory;
  • using different types of paper trays for special paper sizes;
  • using only new paper sheets to determine any pre-existing problem with the print head.

03. Which two scanner options are used to improve document or image scans? (Choose two)

  • resolution changes;
  • ouput file format;
  • printing capacity;
  • source image size ;
  • input file format.

04. Which two scanning settings can significantly affect the size of the image file? (Choose two)

  • resolution;
  • brightness;
  • sharpening;
  • color inversion;
  • color correction;
  • output file format.

05. Which Windows component allows the Windows operating system to output graphics to a computer screen or print device?

  • GDI;
  • GUI;
  • MDL;
  • WPL;
  • WPS.

06. Which two items can help extend the life of an inkjet printer? (Choose two)

  • driver version;
  • type of ink;
  • type of operating system;
  • paper type;
  • type of print server;
  • type of cable to connect to the PC.

07. A technician is installing network printer software and drivers on a Windows Vista computer. At the Select a Printer stage the technician wants to install a printer that is not listed.Which two methods are available to identify and select the required printer ? (Choose two)

  • name;
  • location;
  • IP address;
  • Printer type;
  • Manufacturer. 

08. Which two methods are often used to connect a lower-cost printer to a Windows XP network? (Choose two)

  • use a separate print server;
  • download a network-compliant driver;
  • update to a networking firmware;
  • connect to a networked computer and share the printer;
  • install a NIC to the printer;
  • install a Bluetooth adapter in the printer. 

09. For a user or technician performing maintenance on a dot matrix printer, what potential safety hazard does a print head present?

  • cuts from sharp blades;
  • burns from hot surfaces;
  • illness from toxic chemicals;
  • electrocution from high voltages.

10. Which type of port is used for low-speed dot matrix printers?

  • Ethernet;
  • FireWire;
  • SCSI;
  • Serial;
  • USB.

11. What can a technician do to reduce the time a user must wait when large print jobs are sent to the printer?

  • Change to a networking operating system;
  • Upgrade the operating system;
  • Add more RAM to the printer;
  • Change to a newer version of the firmware;
  • Reinstall the print driver.

12. Which printer component can be upgraded to improve the buffering capabilities of the printer?

  • Duplexer;
  • Memory;
  • Special media tray;
  • Wireless network card.

13. Which two printer components can retain high voltages even after printer has been unplugged?

  • corona wire;
  • transfer roller;
  • toner cartridge;
  • thermal inkjet nozzle;
  • dot matrix print head;
  • piezoelectric inkjet nozzle.

14. Which path will allow a user to add a printer in Windows XP?

  • Start>Control Panel>Add/Remove Hardware>Add Printer;
  • Start>Control Panel>Printers and Faxes>Find a Printer;
  • Start>System>Printers and Faxes>Add a Shared Printer;
  • Start>Control Panel>Printers and Faxes>Add a Printer.

15. A technician needs to set up multiple printers in a office so that multiple users can access the printers without using the resources of any client computers. What is the best solution to meet these needs?

  • computer shared printers;
  • directly attached printers;
  • a dedicated PC print server;
  • a print spooler service enabled on each PC. 

16. What two solutions would fix problem that occurs when every page that is printed by a laser printer produces lines or streaks ? (Choose two)

  • replace the drum;
  • update the printer driver;
  • set the printer to the Use Printer Online option;
  • remove and shake the toner cartridge;
  • use the cleaning utility to clean the cartridge nozzles.

17. A technician is trying to allow user access to a printer on a small network. Which solution is the lowest in cost?

  • wired print server;
  • dedicated PC print server;
  • computer-shared printer;
  • wireless print server;
  • on-demand PC print server. 

18. Users from the art department are complaining that the inkjet printer is producing spolty and streaked printouts. What can be recommended to resolve this problem?

  • restore the printer quality to normal setting;
  • replace the print head;
  • check the printer manual for RAM specifications;
  • clean the print head by running the cleaning utility.
