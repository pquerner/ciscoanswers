Chapter Windows 7  – IT Essentials

01. A PC that running the Windows 7 operating system has slowed down over a period of time and now takes a long time to launch applications and open various documents. What can be done to remedy the problem?

  • Run CHKDSK
  • Run Dr.Watson
  • Run System Restore
  • Run Disk Defragmenter

02. After using a Windows7 installation DVD to upgrade a PC from Windows Vista to Windows7, a user finds that the DVD drive is no longer recognized. What is the probable cause of the problem?

  • The Windows registry cannot initialize;
  • The file system does not support the DVD drive;
  • The UAC setting is too high for basic computing;
  • A registry error prevents Windows 7 from recognizing the DVD drive.

03. A user notices that some of the programs that are installed before upgrading to Windows7 no longer function properly. What can the user do to fix this problem?

  • Change the file system to FAT16;
  • Update the driver for the graphics card;
  • Reinstall the programs in Compatibility Mode;
  • Lower the UAC setting in the Change User Account Control settings dialog box of the User Accounts control.

04. Which Windows7 feature allows you to view the contents of the desktop by moving the mouse to the right edge of the taskbar?

  • Aero Peek
  • Snap
  • Shake
  • Search
  • Jump lists
  • Thumbnail previews

05. How is the Microsoft Assessment and Planning toolkit software used in the upgrade to Windows7?

  • It is used to deploy automated instalations of Windows7 to networked computers;
  • It is used to gather file system compatibilities of computers prior to Windows7 upgrades;
  • It is used to assess network conditions for a successful network deployment of Windows7;
  • It is used to assess existing computers to determine whether they can be migrated to Windows7.

06. A user notices that the Windows 7 UAC appears too often after upgrading the computer to Windows7. What can the user do to correct this problem?

  • Reinstall all user programs in Compatibility Mode;
  • Replace the graphics card with one that meets the minimum hardware requirements for Windows7;
  • Lower the UAC setting in the Change User Account Control Settings dialog box of the User Accounts control;
  • Choose Adjust for Best Performance in the Performance Options dialog box of the Performance Information and Tools control panel.

07. A technician is planning an operating system upgrade for a user system that is currently running Windows Vista. Because the BitLocker feature embedded in the Windows Vista OS is required, there is a request that this feature be available when the new operating system is installed. Which version or versions of Windows 7 would support this feature?

  • Windows 7 Ultimate;
  • Windows 7 Ultimate and Professional;
  • Windows 7 Ultimate and Professional and Home Premium;
  • Windows 7 Ultimate and Professional and Home Premium and Starter.

08. A technician has created a partion in preparation for a Windows 7 network installation on a PC. What additional step is required to allow the PC to connect to the file server to complete the installation?

  • The technician must format the file system on the local PC with NTFS;
  • The technician must copy the Windows 7 installation media to the local PC;
  • The technician must make the partition bootable and include a network client on the local PC;
  • The technician must set up a separate NTFS partition on the local PC and apply full access permissions to the partition.

09. A new Windows7 user has used the Shake feature to effectively control the desktop. After moving to another system which also emplys Windows7, the user notices that the Shake feature does not work. What could be a possible problem?

  • There is not enough RAM to support this feature;
  • The computer is using Windows 7 Starter edition;
  • Only the Ultimate edition of Windows 7 supports this feature;
  • The Windows 7 Professional edition that is installed on the computer is an upgrade from Windows XP and does not support this feature.

10. Which two options correctly list the amount of hard drive space that is required for a Windows7 Professional installation on 32-bit and 64-bit systems? (choose two)

  • 6GB hard drive space for the 32-bit version;
  • 1GB hard drive space for the 32-bit version;
  • 15GB hard drive space for the 64-bit version;
  • 20GB hard drive space for the 64-bit version;
  • 16GB hard drive space for the 32-bit version;

11. A user on a Windows 7 Home Premium computer reports that the Aero theme does not work. What action could the technician take to resolve this issue?

  • Use the Task Manager to restart the Aero program;
  • Lower the User Account Control settings to a level which will allow Aero to load correctly;
  • Replace the graphics card with one that meets the minimum hardware requirements for Windows7;
  • Replace the Windows 7 Home Premium operating system with a Windows 7 Ultimate operating system.

12. Which operating systems can be upgraded to Windows 7?

  • Windows Vista
  • Windows Vista and Windows XP
  • Windows XP and Windows 2000
  • Windows XP, Windows NT and Windows 2000

13. After a computer is powered on, Windows7 fails to start. What initial procedure would a technician use to deal with this failure?

  • Create a System Repair Disc and restore the image;
  • Use the F12 key during the boot processes and start the computer in safe mode;
  • Boot from a Windows 7 installation media and access the System Restore utility;
  • Access the Control Panel and chose Restore Files from Backup from the System and Security menu.

14. A user downloads and uses the Windows7 Upgrade Advisor to generate a report on a PC that is being considered for an operating system upgrade. What information would the report contain?

  • Recommended changes to hardware;
  • Recommended changes to file systems;
  • Recommended changes to user permissions;
  • Recommended changes to user permissions;
  • Recommended changes to directory structures.

15. Which statement about Windows 7 is correct?

  • There is no 64bit version of Windows 7 Home Premium;
  • Windows 7 Starter is only available for netbook computers;
  • All versions of Windows 7 are installed by default in the C:\Windows\folder;
  • Windows 7 Ultimate requires twice as much RAM and hard disk space as Windows 7 Home Premium.

16. A user is trying to run a program in Windows 7, but keeps getting an error message that states that the program must be run with elevated privileges. What should the user do to allow this program to run properly without having an impact on the security of other programs?

  • Turn off UAC;
  • From the Compatibility tab of the Properties of the program, selected Run this Program as an Administrator;
  • Ensure the Administrators group has Full Control permissions in the Security tab of the Properties of the program.
  • In the Compatibility tab of the Properties of the program, choose Run this Program in Compatibility Mode for Windows Vista.

17.  A technician wishes to restart the print spooler on a Windows7 client computer. Which Task Manager tab would provide support for this action?

  • Users;
  • Services;
  • Processes;
  • Networking;
  • Applications.

18. Which Windows7 feature would an administrator use to configure a client computer to perform a CHKDSK procedure at 3:00 AM each day?

  • Disk Cleanup;
  • Task Scheduler;
  • Device Manager;
  • Computer Management.

19. A technician wants to deploy Windows7 to 120 computers in an enterprise. The technician wants to use the least amount of administrative overhead in making sure that all the computers are able to be upgraded successfully. What should the technician do to accomplish this?

  • Run the Windows OS Upgrade Advisor;
  • Use the Microsoft Assessment and Planning Toolkit;
  • Use Windows Anytime Upgrade to upgrade the machine;
  • Insert the Windows 7 installation media and select “Check compatibility online”.

20. Which statement about Windows 7 is correct?

  • Windows7 Professional includes the BitLocker feature;
  • Windows7 Home Premium cannot be joined to a domain;
  • Windows7 Starter supports 64-bit processor arhitectures;
  • Windows7 Ultimate does not include the Aero desktop theme.

21. A technician wants to allow users to backup and restore all the files on their computers, but does not want the users to be able to make changes to the system. How should the technician give users this functionality?

  • Grant the users read permissions on all files;
  • Give the users administrator rights on their computers;
  • Make the users members of a group called Backup Operators;
  • Check the Folder is Ready for Archiving box in the properties of each folder.

22. A Windows 7 computer backup that is scheduled for a specific date and time fails to start. What could be done to address this problem?

  • Choose Turn on Schedule on the Backup and Restore menu;
  • Execute the Startup Repair utility which is available using the System Recovery Options;
  • Ensure that the hard drive has at least 1GB of available space for storing the entire backup file;
  • Choose System Restore from the System Recovery Options and set the scheduler time.

23. How should a hard drive be partitioned to support a Windows7 installation over a network?

  • A FAT 16 partition of least 2GB must be created;
  • A FAT 32 partition of least 5GB must be created;
  • An NTFS partition of least 2GB must be created;
  • An NTFS partition of least 3GB must be created.

24. A technician has determined that a cache overflow problem is having an impact on the performance of a Windows7 computer. How could the technician correct this problem?

  • Open an IE browser window and enable InPrivate Filtering;
  • Change the system environment variables by using the path Start> Control Panel> System and Security> System;
  • Change the environment variables stored at C:\Documents and Settings\%USERPROFILE%\Local Security> System;
  • Change the size of the cache by opening an IE browser window, navigating to Tools>Internet Options, and clicking Settings in the Browsing History area.

25. A user decides to install the Windows7 operating system on the hard drive of a PC using the default settings. The user creates a single partition using the entire space on the hard drive. What file system will the partition be formatted with during the install?

  • NTFS
  • HPFS
  • FAT16
  • FAT32

26. What does %USERPROFILE% represent in the path C:\Users\%USERPROFILE%\AppData\Local\Temp?

  • The location of the Documents folder of a particular user;
  • An environment variable that is set by the operating system;
  • A variable set by the administrator for each user of the system;
  • A fixed value that represents the level of access of a user to system files.

27. A Windows7 operating system fails to boot when the system is powered on. The technician suspects that the operating system has been attacked by a virus rendered the system inoperable. What measure could be taken to restore system functionality?

  • Use a differential backup to restore the system;
  • Use an incremental backup to restore the system;
  • Use a system image that was created prior to the failure to restore the system;
  • Run chkdsk on the affected drive to restore system files to their post-installlation state.

28. A technician is configuring a kiosk PC in the office lobby for customer use. This kiosk needs to be able to display information in English, French and Spanish. The technician has decided to use Multilingual User Interface packs with Windows7 to provide the capability of switching easily between languages. What is the minimum version of Windws7 the technician can use to support this feature?

  • Windows7 Starter;
  • Windows7 Ultimate;
  • Windows7 Professional;
  • Windows7 Home Premium.

29. In wich folder are application files for 32-bit programs typicaly located on a computer that is running a 64-bit edition of Windows7 ?

  • C:\Users;
  • C:\Program Files;
  • C:\Aplication Data;
  • C:\Program Files (x86).

30. A technician is trying to configure offline files and folders for a Windows7 Home Premium desktop machine, but cannot get the feature to work. What is the most likely cause of the problem?

  • The C:\Windows\CSC folder does not exist;
  • The PC has lost its connection to the network;
  • The current user does not have the correct permissions on the files;
  • Windows 7 Home Premium does not support offline files and folders.

31. Which feature of Windows7 allows easy access to common tasks specific to a connected device, such as synching or checking for updates?

  • Device Stage;
  • Task Manager;
  • Device Manager;
  • Computer Management.

32. A network technician is deploying Windows7 and has been asked to ensure that all machines can join the domain use jump lists, have location aware printing, and use BitLocker to secure data. What is the minimum version of Windows 7 that the technician can deploy to meet all these requirements?

  • Windows7 Starter;
  • Windows7 Ultimate;
  • Windows7 Professional;
  • Windows7 Home Premium.
